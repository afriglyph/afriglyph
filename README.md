# AfriGlyph Website

The marketing website for AfriGlyph. It’s a static site built with Astro — landing page plus a couple of supporting pages.

Pushing to `main` triggers a GitHub Actions workflow that builds and deploys the site to GitHub Pages.

## Pages

**Home** — Main landing page with product overview, how it works, in-app preview, glyph library, verified attribution, and pilot section.

**Creator attribution** — Example artist attribution page.

**Privacy** — Privacy policy page (placeholder copy for now).

## Local preview

```bash
npm install
npm run dev
```

Open [http://localhost:4321](http://localhost:4321).

## Reference files

The `Resources/` folder holds the original prototype and reference materials. Those files are not deployed with the site.
